# ROOTS - Farmer-Retailer Trading Platform

ROOTS is a professional web application designed to bridge the gap between farmers and retail buyers. It provides a secure, transparent, and efficient marketplace for agricultural trade.

### üöÄ Features
- **Secure Authentication**: User registration with email OTP verification.
- **Role-Based Access**: Specialized dashboards for Farmers, Retailers, and Admins.
- **Inventory Management**: Farmers can list products, manage quantity, and track sales.
- **Seamless Ordering**: Retailers can browse products, manage a cart, and place orders.
- **Integrated Payments**: Secure payment processing via Razorpay.
- **Automated Documentation**: Dynamic PDF invoice generation and automated email notifications (Welcome, Order Placed, Payment Success).
- **Communication**: Real-time notification system and integrated complaint/feedback module.
